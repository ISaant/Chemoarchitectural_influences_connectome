# Chemoarchitectural Influences on the Cortical Connectome

This repository contains the analysis pipeline used to study the relationship between cortical chemoarchitecture and frequency-resolved functional connectivity derived from MEG recordings.

The workflow integrates:
- MEG source-reconstructed data
- Amplitude Envelope Correlation (AEC) functional connectivity
- PET-derived neurotransmitter receptor/transporter atlases
- Morphometric variables
- Linear mixed-effect models
- Leave-one-out neurotransmitter analyses

The analyses were developed to investigate how neurochemical similarity constrains large-scale cortical communication across frequency bands and aging-related trajectories.

---

# Repository Structure

## Main Scripts

| Script | Description |
|---|---|
| `AEC_plus_morphing.py` | Creates Amplitude Envelope Correlation (AEC) matrices from source-reconstructed MEG data. |
| `create_neuromaps_csv.py` | Extracts and organizes neurotransmitter maps into CSV format. |
| `create_neuromaps_correlation_matrix.py` | Generates neurotransmitter similarity matrices from the CSV generated by `create_neuromaps_csv.py`. |
| `create_neuromaps_leaveOneOut_correlation_matrix.py` | Generates leave-one-out neurotransmitter similarity matrices. |
| `create_main_table.py` | Integrates subject metadata, connectivity metrics, and neurotransmitter similarity outputs into a final analysis table. |
| `*.R` | Linear mixed-effect model analyses and statistical modeling. |

# Pipeline Overview

## 1. Generate Parcellation Labels

The analyses use the Schaefer 2018 200-parcel, 17-network atlas projected from `fsaverage` to each participant.

Template:
https://github.com/ThomasYeoLab/CBIG/tree/master/stable_projects/brain_parcellation/Schaefer2018_LocalGlobal/Parcellations/FreeSurfer5.3/fsaverage

For each subject:

```bash
SUBJ=your_subject

# Left hemisphere
mri_surf2surf --srcsubject fsaverage \
  --trgsubject $SUBJ \
  --hemi lh \
  --sval-annot $SUBJECTS_DIR/fsaverage/label/lh.Schaefer2018_200Parcels_17Networks_order.annot \
  --tval $SUBJECTS_DIR/$SUBJ/label/lh.Schaefer2018_200Parcels_17Networks_order.annot

# Right hemisphere
mri_surf2surf --srcsubject fsaverage \
  --trgsubject $SUBJ \
  --hemi rh \
  --sval-annot $SUBJECTS_DIR/fsaverage/label/rh.Schaefer2018_200Parcels_17Networks_order.annot \
  --tval $SUBJECTS_DIR/$SUBJ/label/rh.Schaefer2018_200Parcels_17Networks_order.annot
```

## 2. Generate AEC Matrices

AEC_plus_morphing.py creates Amplitude Envelope Correlation matrices.

Requirements:

- Source-reconstructed MEG data
- Subject-specific Schaefer labels generated with mri_surf2surf
- FreeSurfer subjects directory properly configured

Outputs:

- Frequency-specific AEC matrices
- Morphometric information (if enabled)

## 3. Generate Neurotransmitter Maps

Run: python create_neuromaps_csv.py
This script creates the neurotransmitter CSV file used in downstream analyses.

## 4. Generate Neurochemical Similarity Matrices

Run: create_neuromaps_correlation_matrix.py
This script waits for the output generated by: create_neuromaps_csv.py

Outputs:
* Neurochemical similarity matrices
* Pairwise neurotransmitter correlations across cortical parcels

## 5. Generate Leave-One-Out Neurochemical Matrices

Run: create_neuromaps_leaveOneOut_correlation_matrix.py
This generates neurotransmitter similarity matrices while excluding one neurotransmitter system at a time.

Outputs:
* Leave-one-out neurochemical similarity matrices
* Sensitivity analyses for neurotransmitter contributions

## 6. Create Main Analysis Table

Run: create_main_table.py
This script integrates all derived metrics into a single table for statistical modeling.

### Required Inputs
#### Subject Metadata CSV

A CSV containing: 
* Subject IDs in the first column
* IDs formatted as: sub-CCXXXX
* Age
* Sex
* Cognitive scores
* Clinical variables
* Behavioral variables
* Covariates

#### Required Outputs:
The script also requires outputs generated by:

* create_neuromaps_correlation_matrix.py
* AEC_plus_morph.py
* create_neuromaps_leaveOneOut_correlation_matrix.py

Outputs:

* Final analysis table for statistical modeling
* Integrated neurochemical, functional connectivity, and subject-level variables

## Statistical Modeling

All statistical analyses are implemented in the R scripts included in this repository.

These scripts primarily perform:

* Linear mixed-effect modeling: Fixed-effect / Random-effect analyses
